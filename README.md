2. Express

4. Socket.io

To use it simply make .env file with values as :
```
DATABASE_USER = <USERNAME>
DATABASE_HOST = <localhost> / <IP ADDRESS>
DATABASE_NAME = <DATABASE NAME>
DATABASE_PASSWORD = <DATABASE PASSWORD
DATABASE_PORT = <DATABASE PORT>
PORT = <PORT>
```
